Memory Orchestrator (Root Agent)

系统核心中枢，统一调度所有功能代理的工作流程

Ingest Agent

  • Summarize（内容总结）
  • Extract entities（实体提取）
  • Tag topics（主题打标）

Consolidate Agent

  • Find patterns（发现模式）
  • Merge related memories（合并相关记忆）

Query Agent

  • Search memories（搜索记忆）
  • Synthesize answers（合成答案）

SQLite DB (memory.db)

所有记忆数据的统一持久化存储介质，承接Ingest、Consolidate、Query代理的处理结果，为整个系统提供数据支撑
